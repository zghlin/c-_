<hr>
# 标题
要创建标题，须在文本前加1到6个#,#的数量影响标题的大小与格式。<br>
如
```
# the first thing
## the first thing
### the first thing
#### the first thing
##### the first thing
###### the first thing
```

# the first thing
## the first thing
### the first thing
#### the first thing
##### the first thing
###### the first thing

## 斜体
```
*斜体*
_斜体_
```

*斜体*<br>
_斜体_<br>

## 删除线
```
~~删除线~~
```

<hr>
## 代码块
六个 ~ 或 `，三个一组，代码块内的文本不会被格式化
~~~
123
124
~~~
```
this is codeblocks
```
代码块可标明所使用的编程语言进行高亮显示
如
```c
printf("hello,world");
```
单行代码还可以这样写
```
` printf("hello,world");
```
输出效果为<br>
` printf("hello,world");`
<hr>
